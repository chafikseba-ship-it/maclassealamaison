a² Identités remarquables

La leçon — étape 1 sur 3
📐 Il y a 3 identités remarquables à connaître par cœur !

① Carré de somme : (a+b)² = a² + 2ab + b²
② Carré de différence : (a−b)² = a² − 2ab + b²
③ Produit de conjugués : (a+b)(a−b) = a² − b²
Moyen mnémotechnique :
somme · différence · conjugués

⚠️ Erreur classique : (x+3)² ≠ x²+9 — il ne faut pas oublier le terme 2ab !

La leçon — étape 2 sur 3
🎯 Exemple : Développe (x+3)²

On identifie : a = x, b = 3
(a+b)² = a² + 2ab + b²
= x² + 2×x×3 + 3²
= x² + 6x + 9
ÉcritureRésultat
(x+3)²x²+6x+9 ✅
(x+3)² ≠x²+9 ❌
La leçon — étape 3 sur 3
🎯 Exemple : Développe (2x−1)²

On identifie : a = 2x, b = 1
(a−b)² = a² − 2ab + b²
= (2x)² − 2×2x×1 + 1²
= 4x² − 4x + 1
🏆 Identité ③ : (x+4)(x−4) = x² − 4² = x² − 16
On reconnaît (a+b)(a−b) = a²−b² avec a=x, b=4.

📖 Développer

La leçon — étape 1 sur 2
📖 Distributivité simple : a(b+c) = ab + ac

Double distributivité : (a+b)(c+d) = ac + ad + bc + bd

Règle FOIL :
First (premiers) · Outer (extérieurs) · Inner (intérieurs) · Last (derniers)
3(x+2) = 3x + 6
La leçon — étape 2 sur 2
🎯 Exemple : Développe (3x+2)(x−4)

F : 3x × x = 3x²
O : 3x × (−4) = −12x
I : 2 × x = +2x
L : 2 × (−4) = −8

= 3x² − 12x + 2x − 8
= 3x² − 10x − 8 (termes semblables réduits)
  • Distribuer le premier facteur sur chaque terme du second
  • Écrire tous les produits
  • Réduire les termes semblables (en x)

🔧 Factoriser

La leçon — étape 1 sur 2
🔧 Factoriser = mettre en facteur commun.

Méthode : trouver le PGCD des coefficients et le plus petit exposant de x.

Exemple : 6x + 4
PGCD(6,4) = 2 → on met 2 en facteur
= 2(3x + 2)
6x² + 9x = 3x(2x + 3)

Vérification : 3x × 2x = 6x² et 3x × 3 = 9x ✅

La leçon — étape 2 sur 2
🔍 Reconnaître les identités à l'envers :

x² − 16 = x² − 4² = (x+4)(x−4)
(identité ③ à l'envers)

x² + 6x + 9 = x² + 2×3×x + 3² = (x+3)²
(identité ① à l'envers)
a²−b² → (a+b)(a−b)
a²+2ab+b² → (a+b)²
a²−2ab+b² → (a−b)²

√ Racines carrées

La leçon — étape 1 sur 2
📐 Définition : √a = nombre positif dont le carré est a.
√a ≥ 0 toujours !

Exemples : √4 = 2, √9 = 3, √25 = 5, √2 ≈ 1,414

Propriétés :
√(ab) = √a × √b
√(a/b) = √a / √b
(√a)² = a   et   √a × √a = a
La leçon — étape 2 sur 2
🔬 Simplification : chercher un facteur carré parfait.

√12 = √(4×3) = √4 × √3 = 2√3
√50 = √(25×2) = √25 × √2 = 5√2
√18 = √(9×2) = 3√2
  • Chercher le plus grand carré parfait diviseur
  • Écrire le produit sous le radical
  • Sortir la racine entière du radical

≤ Inéquations

La leçon — étape 1 sur 2
⚖️ On résout une inéquation comme une équation, SAUF :

🚨 Règle d'or : si on multiplie ou divise les deux membres par un nombre NÉGATIF, le sens de l'inégalité change !

2x > 6 → x > 3   (divisé par +2, sens inchangé)
−2x > 6 → x < −3   (divisé par −2, sens inversé !)
÷ ou × par négatif → inégalité se retourne !
La leçon — étape 2 sur 2
📝 Notation intervalle de la solution :

x > 3 → x ∈ ]3 ; +∞[
x ≤ −2 → x ∈ ]−∞ ; −2]
−1 < x ≤ 5 → x ∈ ]−1 ; 5]

Crochet ] ou [ = valeur exclue
Crochet [ ou ] = valeur incluse (avec ≤ ou ≥)
] exclut la borne
[ inclut la borne
+∞ et −∞ toujours exclus
