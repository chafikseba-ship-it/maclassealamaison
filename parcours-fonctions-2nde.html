f(x) — Notation

La leçon — étape 1 sur 3
Trois mots clés à retenir :
f est la fonction. x est la variable (ou l'antécédent). f(x) est l'image de x.

Notation : f : x ↦ f(x). Exemple : f : x ↦ 2x + 1.
"f(x) = 2x + 1" se lit : "f de x égale deux x plus un".
f : x ↦ 2x + 1

La flèche ↦ se lit "associe à". x est l'entrée, f(x) est la sortie.

La leçon — étape 2 sur 3
Calculer f(3) avec f(x) = 2x + 1 : on remplace x par 3 partout.

f(3) = 2 × 3 + 1 = 6 + 1 = 7

Donc 7 est l'image de 3 par f.

La leçon — étape 3 sur 3
Le domaine de définition = ensemble des valeurs de x autorisées.

• f(x) = 2x + 1 : x peut être n'importe quel réel → domaine = ℝ
• f(x) = 1/x : x ne peut pas valoir 0 → domaine = ℝ \ {0}
• f(x) = √x : x doit être positif → domaine = [0 ; +∞[
f(x) = 2x+1 → ℝ
f(x) = 1/x → x ≠ 0
f(x) = √x → x ≥ 0

📥 Calculer une image

La leçon — étape 1 sur 2
Méthode : f(x) = 3x² − 1. Calculer f(2) : remplacer x par 2.

f(2) = 3 × (2)² − 1
= 3 × 4 − 1
= 12 − 1
= 11
Règle : TOUJOURS remplacer x par la valeur, avec des parenthèses.
La leçon — étape 2 sur 2
⚠️ Attention aux nombres négatifs : f(x) = x² + 1.

f(−3) = (−3)² + 1 = 9 + 1 = 10

(−3)² = 9, pas −9 ! Les parenthèses sont obligatoires.
(−3)² = 9 ✓
−3² = −9 ✗

🔎 Trouver un antécédent

La leçon — étape 1 sur 2
"Antécédent de 7 par f" = trouver x tel que f(x) = 7.

C'est l'opération inverse du calcul d'image.
Image : on donne x, on trouve f(x).
Antécédent : on donne f(x), on cherche x.
f(x) = k → résoudre l'équation
La leçon — étape 2 sur 2
f(x) = 2x + 1. Antécédent de 7 ?

On pose f(x) = 7 : 2x + 1 = 7
2x = 7 − 1 = 6
x = 6 ÷ 2 = 3

Vérification : f(3) = 2 × 3 + 1 = 7 ✓
  • Poser f(x) = k (la valeur donnée)
  • Isoler x en résolvant l'équation
  • Vérifier en recalculant f(x)

📈 Sens de variation

La leçon — étape 1 sur 2
f est croissante sur I : si x₁ < x₂ dans I, alors f(x₁) < f(x₂).
Quand x augmente, f(x) augmente. Flèche ↗

f est décroissante sur I : si x₁ < x₂, alors f(x₁) > f(x₂).
Quand x augmente, f(x) diminue. Flèche ↘
↗ Croissante
↘ Décroissante
La leçon — étape 2 sur 2
Pour une fonction affine f(x) = ax + b :
• Si a > 0 → f est croissante sur ℝ
• Si a < 0 → f est décroissante sur ℝ

Exemple : f(x) = 2x + 3 → a = 2 > 0 → croissante
Exemple : f(x) = −3x + 1 → a = −3 < 0 → décroissante
Signe de a = sens de variation

📊 Tableau de valeurs

La leçon — étape 1 sur 2
Un tableau de valeurs donne des exemples de couples (x, f(x)).

Méthode : calculer f(x) pour chaque valeur de x choisie.
On remplace x par chaque valeur, une par une.
Pour chaque x → calculer f(x)
