📖 Vocabulaire du calcul littéral

🧲 Les briques de base

Une expression littérale contient des lettres (variables) et des nombres. Exemple : 3x + 5

Un monôme est un seul terme : produit d'un nombre et d'une variable.

Exemples de monômes : 4x², −3y, 7

Un polynôme est une somme de monômes.

Binôme (2 termes) : 2x + 3
Trinôme (3 termes) : x² + 2x − 1
🎓 Degré et coefficient

Le degré d'un polynôme = l'exposant le plus élevé de la variable.

2x³ − x + 1 → degré 3
x² + 5x + 6 → degré 2

Le coefficient = le nombre devant la variable.

Dans 5x, le coefficient est 5.
Dans −3x², le coefficient est −3.

📌 Un nombre seul (ex : 7) est un polynôme de degré 0 — on dit une constante.

📐 Développer une expression

🔵 Distributivité simple

k⋅(ax + b) = k⋅ax + k⋅b

On multiplie chaque terme de la parenthèse par le facteur extérieur.

3⋅(2x + 5)
= 3⋅2x + 3⋅5
= 6x + 15
🔸 Distributivité double

(ax + b)⋅(cx + d) = ac⋅x² + (ad + bc)⋅x + bd

On fait 4 produits (chaque terme à gauche fois chaque terme à droite), puis on regroupe.

(x + 2)⋅(x + 3)
= x⋅x + x⋅3 + 2⋅x + 2⋅3
= x² + 3x + 2x + 6
= x² + 5x + 6

💡 Le petit point ⋅ est le signe de multiplication — à ne pas confondre avec la variable x.

🗣 Identités remarquables

⭐ Les 3 identités à connaître par cœur !
1. (a + b)² = a² + 2ab + b²
Preuve : (a+b)⋅(a+b) = a⋅a + a⋅b + b⋅a + b⋅b = a² + 2ab + b²
(x + 3)² = x² + 2⋅x⋅3 + 3² = x² + 6x + 9
2. (a − b)² = a² − 2ab + b²
Attention : le signe moins change le milieu mais PAS les extrémités !
(x − 4)² = x² − 2⋅x⋅4 + 4² = x² − 8x + 16
3. (a + b)(a − b) = a² − b² ← "différence de carrés"
Les termes du milieu se simplifient ! +ab − ab = 0
(x + 5)(x − 5) = x² − 5² = x² − 25
📋 Méthode
  1. Identifie a et b dans l'expression
  2. Choisis la bonne identité parmi les 3
  3. Remplace et calcule les coefficients

✂️ Réduire une expression

📌 Règles des parenthèses
+ (ax + b) = +ax + b ← signe + : on garde tout
− (ax + b) = −ax − b ← signe − : on change tous les signes !

⚠️ Le piège : −(x + 5) = −x − 5, et non −x + 5 !

💡 Exemples guidés

Exemple 1 — Somme :

3(x + 2) + 2(x − 1)
= 3x + 6 + 2x − 2 ← on développe
= (3x + 2x) + (6 − 2) ← on regroupe
= 5x + 4

Exemple 2 — Différence :

4(x + 3) − (x + 5)
= 4x + 12 − x − 5 ← attention ! −(x+5) = −x − 5
= (4x − x) + (12 − 5)
= 3x + 7

🤱 Factoriser une expression

🔍 C'est quoi factoriser ?

Factoriser = écrire comme un PRODUIT de facteurs (avec des parenthèses). C'est l'opposé de développer !

🔵 Méthode 1 — Facteur commun

On cherche le plus grand nombre qui divise tous les coefficients.

6x + 4 = 2⋅(3x + 2) ← le facteur commun est 2
15x² − 10x = 5x⋅(3x − 2) ← le facteur commun est 5x
🔸 Méthode 2 — Identités (lire à l'envers !)
x² − 9 = x² − 3² = (x+3)(x−3)
x² + 6x + 9 = (x+3)²
x² − 8x + 16 = (x−4)²
