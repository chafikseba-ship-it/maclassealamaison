Substituer

La leçon

Substituer, c’est remplacer la lettre x par un nombre, puis calculer. On respecte les priorités (puissances, ×, ÷, puis + et −).

Exemple : pour x = 4, l’expression 3x + 5 vaut 3 × 4 + 5 = 17.

Réduire

La leçon

Réduire, c’est regrouper les termes semblables : les « x » avec les « x », les « x² » avec les « x² », et les nombres entre eux. On n’additionne jamais des x avec des x².

Exemple : 3x + 5 + 2x − 1 = 5x + 4. Pour écrire un carré dans ta réponse, utilise le bouton x².

Parenthèses

La leçon

Pour supprimer une parenthèse, tout dépend du signe devant :

+ ( … ) : on ne change rien
− ( … ) : on change tous les signes

Exemple : 5x − (2x − 3) = 5x − 2x + 3 = 3x + 3.

Développer

La leçon

Développer, c’est transformer un produit en somme grâce à la distributivité : k(a + b) = k×a + k×b, et (a + b)(c + d) = ac + ad + bc + bd.

Exemples : 3(2x + 5) = 6x + 15 ; (x + 2)(x + 3) = x² + 5x + 6.
