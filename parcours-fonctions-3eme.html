La notion de fonction

Définition

Une fonction, c'est une machine à transformer les nombres : tu lui donnes un nombre, elle te renvoie un seul nombre en retour.

On note la fonction f. Le nombre renvoyé par f quand on lui donne x se note f(x) (on lit « f de x »).

Vocabulaire

Le nombre qu'on donne s'appelle l'antécédent. Le nombre qui sort s'appelle l'image.

Si f(5) = 12 : 12 est l'image de 5, et 5 est un antécédent de 12. L'image entre par la porte « sortie », l'antécédent par la porte « entrée ».

La machine

Exemple avec f(x) = 2x + 3. On entre 4 : la machine multiplie par 2, puis ajoute 3.

Les deux questions à ne pas confondre

Calculer une image = on remplace x (calcul direct). Trouver un antécédent = on résout une équation.

Avec f(x) = 2x + 3 : l'image de 4 → on remplace, f(4)=11. L'antécédent de 11 → on cherche x tel que 2x+3 = 11, donc x = 4.

La fonction linéaire

Définition

Une fonction linéaire s'écrit f(x) = a x, où a est un nombre fixé (le coefficient). La machine se contente de multiplier par a.

Exemple : f(x) = 2x. Alors f(3) = 2 × 3 = 6, et f(−1) = 2 × (−1) = −2.

Proportionnalité

Une fonction linéaire traduit une situation de proportionnalité : les images sont proportionnelles aux antécédents, et a est le coefficient de proportionnalité.

Si 1 stylo coûte 2 €, alors x stylos coûtent f(x) = 2x €. Le prix est proportionnel au nombre de stylos.

Représentation graphique

La courbe d'une fonction linéaire est une droite qui passe par l'origine O(0 ; 0). Le coefficient a donne sa pente.

Pour tracer : la droite passe par O, et par le point (1 ; a). Ici a = 2, donc elle passe par (1 ; 2).

Astuce pourcentages

Augmenter une grandeur d'un pourcentage revient à la multiplier par un coefficient : c'est une fonction linéaire.

Augmenter de 20 % → multiplier par 1,2. Baisser de 30 % → multiplier par 0,7. (100 % + 20 % = 120 % = 1,2)

La fonction affine

Définition

Une fonction affine s'écrit f(x) = a x + b. La machine multiplie par a, puis ajoute b.

Exemple : f(x) = 3x + 2. Alors f(4) = 3 × 4 + 2 = 14.

Les deux nombres clés

a est le coefficient directeur (la pente de la droite). b est l'ordonnée à l'origine : c'est la valeur f(0), là où la droite coupe l'axe vertical.

Pour f(x) = 3x + 2 : a = 3, b = 2. La droite coupe l'axe des ordonnées au point (0 ; 2).

Lien avec la linéaire

Une fonction linéaire est une fonction affine particulière, celle où b = 0. Sa droite passe alors par l'origine.

Calculer un antécédent

Pour trouver l'antécédent de y, on résout l'équation ax + b = y.

f(x) = 2x + 1, antécédent de 7 : 2x + 1 = 7 → 2x = 6 → x = 3. L'antécédent de 7 est 3.

a et b en direct

Joue avec les deux curseurs. a fait pivoter la droite (la pente), b la fait monter ou descendre.

a (pente) = 1
b (origine) = 2
