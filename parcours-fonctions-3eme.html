Vocabulaire des fonctions

La leçon — étape 1 sur 3
🆕 Une fonction, c'est comme une MACHINE magique !
Tu mets un nombre dedans ➜ la machine calcule ➜ tu obtiens un résultat.
x = 3
➔
🛠 Machine
× 2 puis + 1
➔
7

On écrit f(x) = 2x + 1.
Quand x = 3, la machine donne 2 × 3 + 1 = 7.

La leçon — étape 2 sur 3
🎯 On écrit f(3) = 7.
On dit que 7 est l'IMAGE de 3 par f.
La machine a transformé 3 en 7.
x = 3 → on met dans la machine
f(3) = 7 → on obtient 7
7 = IMAGE de 3
La leçon — étape 3 sur 3
🔍 L'ANTÉCÉDENT de 7 par f, c'est 3.
C'est le nombre qu'on a mis dans la machine pour obtenir 7.
f(3) = 7 donc 3 est l'antécédent de 7 par f.
IMAGE : on connaît x, on cherche f(x)
ANTÉCÉDENT : on connaît f(x), on cherche x

Trouver un antécédent

La leçon
🔍 Trouver l'antécédent de 9 par f(x) = 2x + 1.
On cherche x tel que f(x) = 9, c'est-à-dire 2x + 1 = 9.

Méthode pas à pas :

① Pose l'équation : 2x + 1 = 9
② Enlève 1 des deux côtés : 2x = 9 − 1 = 8
③ Divise par 2 : x = 8 ÷ 2 = 4
④ Vérifie : f(4) = 2 × 4 + 1 = 9 ✓

💡 L'antécédent de 9 par f est 4. La calculatrice peut t'aider !

Représentation graphique

La leçon
📈 La courbe représentative de f(x) = 2x + 1 est une droite.
Chaque point de la droite a pour coordonnées (x ; f(x)).
Point (0 ; 1) : f(0) = 2×0+1 = 1
Point (2 ; 5) : f(2) = 2×2+1 = 5

La droite monte parce que le coefficient de x est positif (2 > 0).

💡 Axe horizontal = les x. Axe vertical = les f(x).

Lecture graphique

La leçon
👁 Pour lire f(3) sur le graphique :
1. Cherche x = 3 sur l'axe horizontal
2. Monte verticalement jusqu'à la courbe
3. Lis l'ordonnée sur l'axe vertical ➜ c'est f(3) !
Lire f(3) → point rouge → ordonnée = 7
Antécédent de 7 → point rouge → abscisse = 3

💡 Le point rouge montre où la droite passe en x = 3 : f(3) = 7.

Fonctions linéaires

🔰 La leçon

Une fonction linéaire s'écrit f(x) = a × x

📢 C'est la proportionnalité que tu connais déjà !

Exemple : f(x) = 3x

  • f(1) = 3 × 1 = 3
  • f(2) = 3 × 2 = 6
  • f(5) = 3 × 5 = 15

Le nombre a = 3 s'appelle le coefficient de proportionnalité 📊

📈 La courbe d'une fonction linéaire est une droite qui passe par l'origine (0 ; 0) !

💡 La droite passe toujours par (0 ; 0) — c'est la différence avec les fonctions affines !

🔍 Pour trouver a, tu divises f(x) par x :

Si f(4) = 12 et f(x) = ax, alors :

a = f(4) ÷ 4 = 12 ÷ 4 = 3

Donc f(x) = 3x ✓

Fonctions affines

📈 La leçon

Une fonction affine s'écrit f(x) = ax + b

📢 a = coefficient directeur (la pente) · b = ordonnée à l'origine

Exemple : f(x) = 2x + 1

  • a = 2 (la droite monte de 2 à chaque pas)
  • b = 1 (la droite coupe l'axe y en y=1)
  • f(0) = 2×0+1 = 1 = b ✓
  • f(3) = 2×3+1 = 7

📊 Sur le graphique : b = là où la droite coupe l'axe vertical

Pour trouver a avec 2 points A(x₁;y₁) et B(x₂;y₂) :

a = (y₂ − y₁) ÷ (x₂ − x₁)

Exemple : A(0;1) et B(3;7) → a = (7−1)÷(3−0) = 6÷3 = 2
