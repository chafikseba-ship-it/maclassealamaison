ℝ Les ensembles de nombres

Étape 1 — Les nombres que tu connais
🧠 En collège tu as travaillé avec plein de nombres différents.

Regarde ces exemples :
0, 3, 17, 100 → compter des objets, des années…
−5, −12 → températures négatives, dettes…
1,5 ou ¾ → partager, mesurer…
√2 ≈ 1,414… → longueur de diagonale d'un carré de côté 1
En 2nde, on donne un nom et un symbole à chaque famille.
Étape 2 — L'ensemble ℕ
ℕ = les entiers naturels : 0, 1, 2, 3, 4, …

Ce sont les nombres pour compter. Ils commencent à 0 et montent à l'infini.

✅ 0 ∈ ℕ  ·  7 ∈ ℕ  ·  100 ∈ ℕ
❌ −3 ∉ ℕ  ·  1,5 ∉ ℕ
🧩 Test rapide — étape 3
−3 est-il dans ℕ ?
ℕ = {0, 1, 2, 3, ...}
Étape 4 — L'ensemble ℤ
ℤ = les entiers relatifs : …, −3, −2, −1, 0, 1, 2, 3, …

Tu connais déjà ça depuis la 5ème ! On ajoute les négatifs à ℕ.

✅ −7 ∈ ℤ  ·  0 ∈ ℤ  ·  12 ∈ ℤ
❌ 1,5 ∉ ℤ  ·  ¾ ∉ ℤ
ℕ ⊂ ℤ
Tout entier naturel est aussi un entier relatif
🪆 Poupées russes : ℤ est une grande poupée qui contient ℕ à l'intérieur.
7 ∈ ℕ → donc 7 ∈ ℤ aussi !
Un entier naturel EST un entier relatif (avec un signe + invisible).
🧩 Test rapide — étape 5
0,5 est-il dans ℤ ?
ℤ = {…, −2, −1, 0, 1, 2, …}
Étape 6 — L'ensemble ℚ
ℚ = les rationnels — tous les nombres qui s'écrivent comme une fraction p / q (avec q ≠ 0).

✅ ½ ∈ ℚ  ·  −¾ ∈ ℚ  ·  0,333… = ⅓ ∈ ℚ  ·  5 = 5/1 ∈ ℚ
❌ √2 ∉ ℚ  ·  π ∉ ℚ  (impossible de les écrire comme p/q exact)
ℕ ⊂ ℤ ⊂ ℚ
💡 Décimaux vs rationnels :
— Les décimaux ont une écriture FINIE : 0,5 = ½, −1,25 = −5/4, 3,7 = 37/10.
— ℚ est encore plus grand : il contient aussi 1/3 = 0,333… (infini mais périodique).

🪆 Chaîne : tout entier s'écrit comme fraction → 5 = 5/1 ∈ ℚ, −3 = −3/1 ∈ ℚ.
Un entier est automatiquement rationnel !
🧩 Test rapide — étape 7
√2 est-il dans ℚ ?
ℚ = fractions exactes p/q
Étape 8 — L'ensemble ℝ
ℝ = les réels — TOUS les nombres : rationnels ET irrationnels.

√2 ∈ ℝ  ·  π ∈ ℝ  ·  −7 ∈ ℝ  ·  0,5 ∈ ℝ

Entiers naturelsℕ0, 1, 2, 3…−1, 0,5
Entiers relatifsℤ…−2,−1,0,1,2…0,5, ¾
Rationnelsℚfractions p/q, décimaux, 1/3=0,333…√2, π
RéelsℝTOUS (rationnels + irrationnels)—
🪆 Un entier naturel est aussi entier relatif, rationnel ET réel.
Ex : 3 ∈ ℕ et 3 ∈ ℤ et 3 = 3/1 ∈ ℚ et 3 ∈ ℝ.

∈ Appartenance

Étape 1 — Le symbole ∈
∈ se lit "appartient à".
∉ se lit "n'appartient pas à".

Exemples :
3 ∈ ℕ  (3 est un entier naturel)
−2 ∈ ℤ  (−2 est un entier relatif)
−2 ∉ ℕ  (−2 n'est pas un entier naturel)
Étape 2 — Comment décider ?
Pour savoir si x ∈ E, applique la définition de E :

x ∈ ℕ ? → x est-il un entier positif ou nul ?
x ∈ ℤ ? → x est-il un entier (positif, nul ou négatif) ?
x ∈ ℚ ? → x peut-il s'écrire p/q avec p, q entiers, q ≠ 0 ?
x ∈ ℝ ? → toujours oui (ℝ contient tout)

−8 ∈ ℕ ?
🧩 Test — étape 4
¾ ∈ ℤ ?
🧩 Test — étape 5
π ∈ ℝ ?
π = 3,14159265… (irrrationnel)

⊂ Inclusion

Étape 1 — Le symbole ⊂
⊂ se lit "est inclus dans" (ou "est une partie de").

A ⊂ B signifie : tout élément de A est aussi dans B.

✅ Exemple : ℕ ⊂ ℤ
Tout entier naturel (0, 1, 2, …) est aussi un entier relatif → ℕ est inclus dans ℤ.
❌ Contre-exemple : ℤ ⊄ ℕ
Pourquoi ? −1 ∈ ℤ mais −1 ∉ ℕ.
Il suffit d'UN seul élément de A qui n'est pas dans B pour dire A ⊄ B !
Étape 2 — La chaîne
On a la chaîne complète :

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

Comment retenir l'ordre ? N-Z-Q-R → pense à "NaZiQR" 😄
ℕ ⊂ ℤ
ℤ ⊂ ℚ
ℚ ⊂ ℝ
🪆 Poupées russes : ℝ contient ℚ, qui contient ℤ, qui contient ℕ.
Conséquence : si 7 ∈ ℕ, alors automatiquement 7 ∈ ℤ, 7 ∈ ℚ et 7 ∈ ℝ !
De même, −5 ∈ ℤ → −5 ∈ ℚ et −5 ∈ ℝ.
🧩 Test — étape 3
ℤ ⊂ ℕ ?
Étape 4 — Le symbole ⊄
⊄ se lit "n'est pas inclus dans".

❌ ℤ ⊄ ℕ  (les entiers négatifs ne sont pas dans ℕ)
❌ ℝ ⊄ ℚ  (√2 est réel mais pas rationnel)

💡 Si A ⊂ B, alors tout x ∈ A est automatiquement x ∈ B.
🧩 Test — étape 5
ℚ ⊂ ℝ ?
