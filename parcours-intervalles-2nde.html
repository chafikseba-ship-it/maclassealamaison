📏 Intervalles

Étape 1 — La droite des réels
🧠 Tu connais déjà la droite numérique depuis le collège. Chaque point sur cette droite est un nombre réel.

Les réels comprennent les entiers, les décimaux, et même des nombres comme √2 ou π.

En 2nde, on va apprendre à désigner des portions de la droite avec une nouvelle notation : ]a, b[ et [a, b].
Tous les points de la droite forment l'ensemble ℝ des réels.
Étape 2 — Intervalle fermé [a, b]
[a, b] = intervalle fermé

Les crochets fermés [ ] signifient que les bornes SONT incluses.

Exemple : [2, 5] contient 2, 3, 4, 5 et tous les réels entre 2 et 5.
✅ 2 ∈ [2, 5]  ·  3,7 ∈ [2, 5]  ·  5 ∈ [2, 5]
❌ 1,9 ∉ [2, 5]  ·  5,1 ∉ [2, 5]
Les points remplis ● indiquent que les bornes sont incluses.
💡 Mémo crochet fermé : le crochet [ ] "serre" la borne, comme une pince — la borne EST prise.
✅ 2 ∈ [2, 5]  ·  ✅ 5 ∈ [2, 5]
🧩 Test rapide — étape 3
3,7 ∈ [2, 5] ?

Étape 4 — Intervalle ouvert ]a, b[
]a, b[ = intervalle ouvert

Les crochets ouverts ] [ signifient que les bornes NE sont PAS incluses.

Exemple : ]2, 5[
❌ 2 ∉ ]2, 5[ (2 est exclu !)
✅ 3,7 ∈ ]2, 5[
❌ 5 ∉ ]2, 5[ (5 est exclu !)
Les cercles vides ○ indiquent que les bornes sont exclues.
❌ Contre-exemple : 2 ∈ [2, 5] MAIS 2 ∉ ]2, 5[
Même borne, résultat opposé ! Le crochet ] [ "repousse" la borne — elle est exclue.
💡 Mémo : ] ouvert = borne dehors  ·  [ fermé = borne dedans
🧩 Test rapide — étape 5
5 ∈ ]2, 5[ ?
]2, 5[ : les crochets ] [ excluent les bornes
Étape 6 — Intervalles mixtes et infinis
On peut aussi mélanger crochets fermés et ouverts :

[a, b[ → a inclus, b exclu  ·  2 ≤ x < 5
]a, b] → a exclu, b inclus  ·  2 < x ≤ 5

Et pour les intervalles infinis, on utilise +∞ et −∞ :
]−∞, a] → tous les réels ≤ a (ex: ]−∞, 3] = x ≤ 3)
[a, +∞[ → tous les réels ≥ a (ex: [2, +∞[ = x ≥ 2)
−∞ et +∞ ont toujours un crochet ouvert
Pourquoi ∞ est toujours ouvert ? Parce que l'infini n'est pas un nombre — on ne peut pas l'atteindre ni l'inclure.

Récap des 6 formes :
[a, b] → a ≤ x ≤ b  ·  ]a, b[ → a < x < b
[a, b[ → a ≤ x < b  ·  ]a, b] → a < x ≤ b
]−∞, b] → x ≤ b  ·  [a, +∞[ → x ≥ a
🧩 Test rapide — étape 7
−1 ∈ ]−∞, 3[ ?
]−∞, 3[ = tous les réels strictement inférieurs à 3

|x| Valeur absolue

Étape 1 — La valeur absolue |x|
|x| = la distance de x à 0 sur la droite numérique.

|3| = 3  (3 est à 3 unités de 0)
|−3| = 3  (−3 est aussi à 3 unités de 0)

💡 La valeur absolue est toujours positive ou nulle. Jamais négative !
📐 Définition précise :
Si x ≥ 0 → |x| = x   (ex : |5| = 5)
Si x < 0 → |x| = −x   (ex : |−3| = −(−3) = 3)

Autrement dit : on enlève le signe négatif. Le résultat est toujours ≥ 0.

|−7| = ?
La valeur absolue = distance à 0
Étape 3 — Calculer |x − a|
|x − a| = la distance entre x et a sur la droite.

Exemple : |x − 2| = distance entre x et 2.

|5 − 2| = |3| = 3  ·  (5 est à 3 unités de 2)
|−1 − 2| = |−3| = 3  ·  (−1 est aussi à 3 unités de 2)
🧩 Test rapide — étape 4
|−4| = ?
|x| = distance de x à 0, toujours ≥ 0
Étape 5 — Lien avec les intervalles
Il y a un lien puissant entre valeur absolue et intervalles :

|x| ≤ 3 signifie : x est à moins de 3 unités de 0
↔ −3 ≤ x ≤ 3
↔ x ∈ [−3, 3]

|x| < 3 ↔ −3 < x < 3 ↔ x ∈ ]−3, 3[
|x| ≤ a ↔ x ∈ [−a, a]
|x| < a ↔ x ∈ ]−a, a[

≤ Inégalités et intervalles

Étape 1 — Appartenance = inégalité double
Écrire x ∈ [2, 7] et écrire 2 ≤ x ≤ 7 c'est la même chose !

Le crochet fermé [ → signe ≤ (inférieur ou égal)
Le crochet ouvert ] → signe < (strictement inférieur)
🔑 L'astuce : regarde le crochet du côté de la borne.
Crochet fermé [ ou ] (pointe vers l'intérieur) → borne incluse → signe ≤
Crochet ouvert ] ou [ (pointe vers l'extérieur) → borne exclue → signe <

]−1, 4[−1 < x < 4
[−3, 5[−3 ≤ x < 5
]0, +∞[x > 0

Étape 3 — Méthode pas-à-pas
Pour savoir si un nombre appartient à un intervalle, remplace par la double inégalité :

Question : 6 ∈ [−3, 5] ?

Étape 1 : [−3, 5] ↔ −3 ≤ x ≤ 5
Étape 2 : Remplace x par 6 : −3 ≤ 6 ≤ 5
Étape 3 : Vrai ou faux ? → 6 ≤ 5 est FAUX
Réponse : 6 ∉ [−3, 5] ❌
🧩 Test rapide — étape 4
4 ∈ ]−1, 4[ ?
]−1, 4[ ↔ −1 < x < 4
