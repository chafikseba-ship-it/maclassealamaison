🪞 Symétrie axiale

La leçon — étape 1 sur 3
🪞 Une symétrie axiale, c'est comme regarder dans un miroir !
L'axe de symétrie, c'est la surface du miroir. Le point A se retrouve de l'autre côté, à la même distance.
La leçon — étape 2 sur 3
📐 Méthode pour trouver A' :
Tu comptes les carreaux de A jusqu'à l'axe. Tu recomptes le même nombre de l'autre côté.
Axe horizontal y = k ➜ A'(x ; 2k − y)
Axe vertical x = k ➜ A'(2k − x ; y)
La leçon — étape 3 sur 3
🎯 Exemple : A(3 ; 2), axe y = 0 (l'axe des x).
L'ordonnée change de signe : A'(3 ; −2).
Si l'axe est x = 0 (l'axe des y) : A'(−3 ; 2).
Axe y = 0 → A'(x ; −y)
Axe x = 0 → A'(−x ; y)

🔵 Symétrie centrale

La leçon — étape 1 sur 3
🔵 Une symétrie centrale, c'est un demi-tour (rotation de 180°) autour d'un point O.
Le point O est le milieu du segment [AA'].
La leçon — étape 2 sur 3
📐 Si le centre est O(0 ; 0) :
A(x ; y) → A'(−x ; −y)

Les deux coordonnées changent de signe !
Centre O(0;0) : A'(−x ; −y)
La leçon — étape 3 sur 3
🎯 Formule générale avec un centre O(a ; b) quelconque :
A'(2a − x ; 2b − y)
Centre O(a;b) : A'(2a − xA ; 2b − yA)

Exemple : O(1 ; 2), A(3 ; 4) → A'(2×1−3 ; 2×2−4) = A'(−1 ; 0)

➡️ Translation

La leçon — étape 1 sur 3
➡️ Une translation, c'est un glissement.
Tous les points se déplacent dans la même direction, du même nombre de cases.
C'est comme faire glisser une pièce de puzzle sans la tourner.
La leçon — étape 2 sur 3
📦 On décrit la translation avec un vecteur t(dx ; dy) :
• dx = déplacement horizontal (+ = droite, − = gauche)
• dy = déplacement vertical (+ = haut, − = bas)
t(3 ; 1) → 3 cases à droite, 1 case en haut
t(−2 ; 3) → 2 cases à gauche, 3 cases en haut
La leçon — étape 3 sur 3
🎯 Formule :
Si A(x ; y) et vecteur t(dx ; dy) :
A'(x + dx ; y + dy)
A'(xA + dx ; yA + dy)

Exemple : A(2 ; 3), t(4 ; −1) → A'(2+4 ; 3+(−1)) = A'(6 ; 2)

🔄 Rotation

La leçon — étape 1 sur 3
🔄 Une rotation, c'est tourner autour d'un point O d'un certain angle.
La distance de A au centre O ne change pas. Seule la direction change.
La leçon — étape 2 sur 3
🔃 Rotation de 180° autour de O(0 ; 0) :
C'est exactement la même chose que la symétrie centrale par O !
A(x ; y) → A'(−x ; −y)
Rot 180° : A'(−x ; −y)
La leçon — étape 3 sur 3
↩️ Rotation de 90° dans le sens direct (sens anti-horaire) autour de O(0 ; 0) :
A(x ; y) → A'(−y ; x)

↪️ Rotation de 90° dans le sens indirect (sens des aiguilles d'une montre) :
A(x ; y) → A'(y ; −x)
90° direct : A'(−y ; x)
90° indirect : A'(y ; −x)

🔍 Homothétie

La leçon — étape 1 sur 3
🔍 Une homothétie, c'est comme zoomer ou dézoomer sur une photo.
On choisit un centre O et un rapport k.
• Si k > 1 : on agrandit
• Si 0 < k < 1 : on réduit
• Si k < 0 : on retourne et on agrandit/réduit
La leçon — étape 2 sur 3
📐 Formule avec centre O(0 ; 0) :
A(x ; y) → A'(k×x ; k×y)

Les deux coordonnées sont multipliées par k.
Centre O(0;0) : A'(k×x ; k×y)

Exemple : k = 2, A(3 ; 1) → A'(6 ; 2) — distances doublées !

La leçon — étape 3 sur 3
🎯 Formule générale avec centre C(a ; b) :
A'(a + k×(xA − a) ; b + k×(yA − b))
A'(a + k(xA−a) ; b + k(yA−b))

Exemple : k = 2, C(1;0), A(3;2) → A'(1+2×(3−1) ; 0+2×(2−0)) = A'(5 ; 4)

Si k = −1 et C = O(0;0) : c'est une symétrie centrale !
