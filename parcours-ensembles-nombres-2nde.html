ℝ Les ensembles ℕ ℤ ℚ ℝ

La leçon — étape 1 sur 3
Il existe 4 grands ensembles de nombres, du plus petit au plus grand :

ℕ = {0, 1, 2, 3, 4, …} — les entiers naturels (positifs ou nuls)
ℤ = {…, −2, −1, 0, 1, 2, …} — les entiers relatifs (positifs et négatifs)
ℚ = fractions a/b (a et b entiers, b ≠ 0) — les rationnels
ℝ = tous les réels (y compris √2, π) — les réels

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ : chaque ensemble est inclus dans le suivant !

La leçon — étape 2 sur 3
Un nombre dans ℕ est aussi dans ℤ, ℚ et ℝ !
Exemple avec 3 :
3 ∈ ℕ
3 ∈ ℤ
3 ∈ ℚ
3 ∈ ℝ
En revanche, −5 :
−5 ∉ ℕ  (pas dans les naturels)
−5 ∈ ℤ  (entier relatif ✓)
−5 ∈ ℚ  (−5/1 est une fraction ✓)
−5 ∈ ℝ  (tout entier est réel ✓)
La leçon — étape 3 sur 3
Les irrationnels sont dans ℝ mais pas dans ℚ !

√2 ≈ 1,4142… (ne s'écrit pas comme fraction a/b) → √2 ∈ ℝ, √2 ∉ ℚ
π ≈ 3,1415… → π ∈ ℝ, π ∉ ℚ

∈ Appartenance ∈ ∉

La leçon — étape 1 sur 2
∈ se lit « appartient à »  |  ∉ se lit « n'appartient pas à »

Exemples :
−3 ∈ ℤ  (−3 est un entier relatif ✓)
−3 ∉ ℕ  (−3 est négatif, pas dans ℕ ✗)
1/2 ∈ ℚ  (c'est une fraction ✓)
1/2 ∉ ℤ  (pas un entier ✗)
√2 ∈ ℝ  (c'est un réel ✓)
√2 ∉ ℚ  (irrationnel ✗)
La leçon — étape 2 sur 2
🗺️ Méthode pour classer un nombre :
  • Est-ce un entier ≥ 0 ? → Il est dans ℕ (et aussi ℤ, ℚ, ℝ)
  • Est-ce un entier négatif ? → Il est dans ℤ\ℕ (et aussi ℚ, ℝ)
  • Est-ce une fraction a/b (non entier) ? → Il est dans ℚ\ℤ (et aussi ℝ)
  • Est-ce √2, π, etc. ? → Il est dans ℝ\ℚ seulement

[ ] Intervalles

La leçon — étape 1 sur 3
Un intervalle est un ensemble de réels entre deux bornes.

[a ; b] : a inclus, b inclus → a ≤ x ≤ b
]a ; b[ : a exclu, b exclu → a < x < b
[a ; b[ : a inclus, b exclu → a ≤ x < b
]a ; b] : a exclu, b inclus → a < x ≤ b

Crochet fermé [ ] = borne incluse (point plein) · Crochet ouvert ] [ = borne exclue (cercle vide)

La leçon — étape 2 sur 3
Les intervalles peuvent être infinis :

]−∞ ; a] : tous les réels ≤ a  (−∞ est toujours exclu !)
[a ; +∞[ : tous les réels ≥ a
]−∞ ; +∞[ = ℝ : tous les réels
x ≤ 3 → x ∈ ]−∞ ; 3]
x > −2 → x ∈ ]−2 ; +∞[
⚠️ On écrit toujours ]−∞ et +∞[ avec des crochets ouverts — jamais [−∞ ou +∞]

|x| Valeur absolue

La leçon — étape 1 sur 2
|x| = la distance de x à 0 sur la droite numérique.

|x| = x si x ≥ 0  (ex : |3| = 3)
|x| = −x si x < 0  (ex : |−4| = −(−4) = 4)
|x| ≥ 0 toujours !
|7| = 7
|−7| = 7
|0| = 0
La leçon — étape 2 sur 2
🎯 Résoudre |x| = k :

Si k > 0 : |x| = k → x = k ou x = −k  (deux solutions)
Si k = 0 : |x| = 0 → x = 0  (une solution)
Si k < 0 : |x| = −1 → pas de solution (|x| ≥ 0 toujours)
|x| = 5 → x = 5 ou x = −5
|x| = 0 → x = 0

⊂ Inclusion

La leçon — étape 1 sur 2
E ⊂ F se lit « E est inclus dans F ».
Cela signifie : tout élément de E est aussi dans F.

Exemple : ℕ ⊂ ℤ car tout entier naturel (0, 1, 2…) est aussi un entier relatif.
ℕ ⊂ ℤ
ℤ ⊂ ℚ
ℚ ⊂ ℝ
La leçon — étape 2 sur 2
🚫 Contre-exemples d'inclusions fausses :

ℤ ⊄ ℕ  car −1 ∈ ℤ mais −1 ∉ ℕ
ℚ ⊄ ℤ  car 1/2 ∈ ℚ mais 1/2 ∉ ℤ
ℝ ⊄ ℚ  car √2 ∈ ℝ mais √2 ∉ ℚ
✅ À retenir : ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ  — dans ce sens seulement !
