Tester une solution

La leçon

Une solution d’une équation est une valeur de l’inconnue qui rend l’égalité vraie. Pour tester une valeur, on la remplace dans les deux membres : si on obtient le même résultat, c’est une solution.

Exemple : pour 3x − 1 = 8 et x = 3 → 3 × 3 − 1 = 8. Vrai → 3 est solution.

Équations du premier degré

La leçon

On regroupe les x d’un côté et les nombres de l’autre, puis on divise. Ce qui passe de l’autre côté du « = » change d’opération (+ devient −, × devient ÷).

Exemple : 3x − 1 = 8 → 3x = 9 → x = 3.

Équation produit nul

La leçon

Un produit de facteurs est nul si, et seulement si, au moins un des facteurs est nul. Donc (x − a)(x − b) = 0 quand x − a = 0 ou x − b = 0.

Exemple : (x − 3)(x + 2) = 0 → x = 3 ou x = −2.

Équation x² = a

La leçon

Si a > 0, l’équation x² = a a deux solutions : √a et −√a. Si a = 0, une seule solution : 0. Si a < 0, aucune solution (un carré n’est jamais négatif).

Exemple : x² = 49 → x = 7 ou x = −7.
